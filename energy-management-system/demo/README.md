## 📈 数据结构

### 三线系统

**精力线（Energy Line）：**
- 初始值：100
- 最终值：85
- 净变化：-15
- 状态：健康 🟢

**时间线（Time Line）：**
- 屏幕使用时间：3.28 小时
- 社交类：120 分钟
- 生产力：60 分钟

**心情线（Mood Line）：**
- 平均值：70-75
- 波动范围：55-85

### 事件类型

**充电事件（+）：**
- 👨‍👧 family - 陪娃、家庭时光
- ☕ rest - 午睡、摸鱼、休息
- ⚡ Aha - 深度对话、吾日三省

**放电事件（-）：**
- 🚧 Friction - 家庭争执、情绪管理
- 🏃 Sprint - 20min 冲刺、重要会议
- 📋 Debt - 碎片任务、日常维护

### 精力汇率规则

**充电：**
- 陪娃：+10
- 午睡：+10
- 吾日三省：+10
- 摸鱼：+5
- 散步：+5

**放电：**
- 家庭争执：-25
- 客户反复沟通：-25
- 重要会议：-20
- 情绪管理：-15
- 20min 冲刺：-10
- 碎片任务：-5

## 💡 核心洞察

### 2026-03-09（周日）

**Top 3 漏电点：**
1. Lawrence 会议（-20，44%）
2. 情绪管理（-15，33%）
3. 碎片任务（-10，22%）

**关键发现：**
- 情绪消耗占比 33%，需要注意
- 午睡 + 看小说充电效果好（+10）
- 陪娃时间充足，家庭充电稳定（+10）

### 2026-03-10（周一上午）

**Top 3 漏电点：**
1. 家庭争执（-25，56%）
2. 20min 冲刺（-10，22%）
3. OpenClaw 笔记整理（-10，22%）

**关键发现：**
- 家庭争执是最大漏电点（56%）
- 吾日三省处理心理问题效果好（+10）
- 摸鱼刷手机是有效充电方式（+5）
- 边走路边交流是舒服的状态（+5）

## 🎯 使用建议

1. **从简单开始** - 先用事件标记，再加入自动采集
2. **坚持 3 天** - 3 天数据就能看到模式
3. **定期复盘** - 每周回顾一次，调整规则
4. **个性化调整** - 根据自己的情况调整汇率规则
