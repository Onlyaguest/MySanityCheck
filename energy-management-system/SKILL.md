# Energy Management System

三线综合的个人精力管理系统。

## 🎯 系统目标

建立一个三线综合的个人状态监控与决策支持系统，帮助你：
- 实时了解自己的精力、时间、心情状态
- 基于当前状态获得任务推荐
- 识别和避免精力漏电点
- 优化日常工作和生活节奏

## 📊 三线系统

### ⚡ 精力线（Energy Line）- 核心

**数据源：**
- 事件标记（#Aha / #Friction / #Sprint）
- 生理数据（睡眠时长、HRV）
- 行为数据（屏幕使用时长、会议频率）

**精力值范围：** 0-100

**精力汇率规则：**
- 充电事件：陪娃 +10、火车留白 +10、深度对话 +5
- 放电事件：#Friction -25~-35、#Sprint -15~-20、#social_drain -25

**状态分级：**
- 80-100：充沛 🟢
- 60-79：良好 🟡
- 40-59：疲劳 🟠
- 20-39：耗竭 🔴
- 0-19：崩溃 ⚫

### ⏰ 时间线（Time Line）- 辅助

**数据源：**
- 日历事件
- 屏幕使用时长
- 应用切换频率

**时间质量评估：**
- ≥2 小时：优质时间（连续）
- 1-2 小时：良好时间
- <1 小时：碎片时间

**时间分配基准：**
- 工作/学习：40-50%
- 生活/家庭：30-40%
- 休息/娱乐：10-20%

### 😊 心情线（Mood Line）- 辅助

**数据源：**
- 主观标记（😊😐😔）
- 日记情感分析
- 行为推断

**心情值范围：** 0-100（默认基准 70）

**起伏基准：**
- 积极事件：深度对话 +10、完成任务 +8、正面反馈 +5
- 消极事件：低效摩擦 -15、家庭争执 -20、工作受挫 -10

## 🎯 综合决策矩阵

| 精力 | 时间 | 心情 | 推荐任务类型 |
|------|------|------|--------------|
| 高 | 多 | 好 | 🚀 高难度创造（系统架构、深度写作） |
| 高 | 多 | 差 | 🔧 机械性工作（代码重构、文档整理） |
| 高 | 少 | 好 | ⚡ 快速冲刺（20min 专注任务） |
| 低 | 多 | 好 | 📚 学习输入（阅读、听播客） |
| 低 | 多 | 差 | 🛌 强制休息（物理熔断） |
| 低 | 少 | 好 | 🎮 轻量娱乐（刷推、看视频） |
| 低 | 少 | 差 | 🚨 紧急干预（停止所有工作） |

## 📈 核心功能

### 1. 晨间校准（Morning Calibration）

**每日 08:00 自动执行**

**输入：**
- 昨晚睡眠质量（Apple Health）
- 今日日程（Calendar）
- 昨日精力消耗（历史数据）

**输出：**
- 今日起始精力值
- 今日可用时间
- 今日任务推荐

### 2. 实时追踪（Real-time Tracking）

**触发时机：**
- 每次事件标记（#Aha / #Friction / #Sprint）
- 自动更新精力值
- 触发预警（精力值 < 30）

### 3. 晚间对账（Evening Review）

**每日 21:00 自动执行**

**输入：**
- 今日所有事件标记
- Screen Time 数据
- Calendar 实际执行情况

**输出：**
- 今日精力消耗总结
- Top 3 漏电点
- 明日调整建议

### 4. 风险预警（Risk Alert）

**自动触发条件：**
- 精力值 < 30：预警精力耗竭
- 时间 < 1 小时：预警时间紧张
- 心情值 < 40：预警情绪低落
- 三线同时低：触发紧急干预

## 💡 核心洞察

### 1. 精力管理是元系统

所有工具都是它的外延：
- Shrimp Factory → 减少部署精力（2h → 5min）
- Agora Reads → 减少信息获取精力
- AI 待机舱 → 优化精力分配
- 微信优化 → 减少隐形漏电

### 2. 三线独立但关联

- 精力线：核心，决定能做什么
- 时间线：辅助，决定能做多久
- 心情线：辅助，决定做得怎么样

### 3. 延迟性精力消耗

- 良性高级交流：当下开心，晚上才累
- 情绪维度 ≠ 精力维度
- 需要分开测量和管理

### 4. 静默收集是关键

- 不为了记录而消耗精力
- 极简按钮 + 自动读取
- 3 天数据就能看到模式

1. **从简单开始** - 先用事件标记，再加入自动采集
2. **坚持 3 天** - 3 天数据就能看到模式
3. **定期复盘** - 每周回顾一次，调整规则
4. **个性化调整** - 根据自己的情况调整汇率规则
5. **不要过度** - 系统本身不应该成为精力消耗源

## ⚠️ 注意事项

### 不做什么
- 不替代专业医疗建议（睡眠障碍、抑郁症等）
- 不强制执行建议（只提供参考）
- 不过度干预（避免成为新的精力消耗源）

### 隐私保护
- 所有数据本地存储（或用户控制的云端）
- 不上传敏感数据（日记内容、聊天记录）
- 用户可随时删除所有数据
