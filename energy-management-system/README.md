# Energy Management System

三线综合的个人精力管理系统。

## 快速开始

查看 [SKILL.md](./SKILL.md) 了解完整功能。

查看 [REQUIREMENTS.md](./REQUIREMENTS.md) 了解详细需求规格。

## 核心概念

- **精力线**：基于事件标记和汇率规则
- **时间线**：基于日历和屏幕使用
- **心情线**：基于主观标记和情感分析

## 使用示例

```bash
# 查询当前状态
查询精力情况

# 标记事件
#Sprint 完成需求文档
#Aha 找到解决方案
#Friction 客户沟通困难
```

## 数据来源

- Apple Health（睡眠、HRV）
- Apple Calendar（日程）
- Screen Time（使用时长）
- 用户标记（事件、心情）

## 输出

- 实时状态查询
- 任务推荐
- 风险预警
- 晨间校准
- 晚间对账
- 周度报告
